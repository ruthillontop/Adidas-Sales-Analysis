# Adidas-Sales-Analysis
Este proyecto consiste en un análisis integral mock up del rendimiento de ventas de Adidas, enfocado en identificar patrones de consumo, rentabilidad por región y eficiencia. El objetivo principal es transformar datos crudos en un Dashboard que facilite la toma de decisiones estratégicas basadas en evidencia.



# Key Insights (Puntos Clave)

Análisis de Rentabilidad: Identificación de las categorías de productos con mayor margen de beneficio.

Desempeño Regional: Comparativa de ventas por ubicación geográfica para detectar oportunidades de expansión.

Canales de Venta: Evaluación de la eficiencia de ventas online vs. tiendas físicas.
